# Web3 Game Assignment

## Tech Stack 

- React@18
- TailwindCSS for styling
- Metamask React SDK 
- React Toastify
- TypeScript
- useContext for global state managememnt

## Installation Instructions

```bash
git clone https://github.com/ayushagarwal27/metamask_assignment.git
```

```bash
npm install
```

```bash
npm run dev
```
